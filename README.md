## рҹ“Ӣ Overview

**SKILLVERSE** is a sophisticated **student-to-student skill exchange platform** that reimagines peer learning through a neobrutalist lens. Built with **vanilla JavaScript**, **Vite 5**, and a custom **reactive state management** system, the platform delivers a seamless single-page application experience across **10 interactive screens** вҖ” from authentication to live chat.

> рҹҺ“ **Academic Project** вҖ” Developed by Team **AC-DC** as part of the VIT Bhopal University curriculum. This project demonstrates advanced frontend engineering patterns including component-based SPA routing, reactive state proxies, real-time AI integration, and a modular CSS design system.

### вңЁ Why SKILLVERSE?

| | |
|---|---|
| **рҹҺҜ Peer-to-Peer Learning** | Connect students with complementary skills for knowledge exchange |
| **рҹӨ– AI-Powered Assistance** | Integrated Groq AI chatbot (Llama 3.1) provides intelligent guidance |
| **рҹҺЁ Neobrutalist Design** | A bold, distinctive visual identity that stands out from conventional UIs |
| **вҡЎ Reactive Architecture** | Custom Proxy-based state management without any frameworks |
| **рҹ“ұ Full User Journey** | 10 interconnected screens covering the complete skill exchange lifecycle |

## вҡЎ Quick Start

### Prerequisites

- **Node.js** 18+ (LTS recommended)
- **npm** 9+ (ships with Node.js)

### Setup in 30 Seconds

```bash
# 1. Clone the repository
git clone https://github.com/Rachit-Tiwari-7/SKILLVERSE.git

# 2. Navigate to project
cd SKILLVERSE

# 3. Install dependencies
npm install

# 4. (Optional) Set Groq API key for AI chatbot features
echo "VITE_GROQ_API_KEY=your_groq_api_key" > .env

# 5. Start development server
npm run dev
```

Your browser will open at **рҹ‘ү [http://localhost:5173](http://localhost:5173)**

### Production Build

```bash
npm run build    # outputs optimized bundle to /dist
npm run preview  # preview production build locally
```
